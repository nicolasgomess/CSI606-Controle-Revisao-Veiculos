Discente: Nicolas de Oliveira Gomes

Resumo



Este projeto tem como objetivo desenvolver um sistema web para controle de revisões de veículos, permitindo que usuários registrem, consultem e gerenciem informações sobre manutenções realizadas e futuras. A aplicação contará com uma interface intuitiva, integração com backend para regras de negócio e banco de dados para armazenar dados como modelo do veículo, quilometragem, tipo de revisão, data prevista e histórico de serviços.

1. Tema



O trabalho final tem como tema o desenvolvimento de um Sistema Web para Controle de Revisões de Veículos, utilizando HTML, CSS, JavaScript (React) no frontend e integração com backend e banco de dados.

2. Escopo



O projeto terá as seguintes funcionalidades:

Cadastro de veículos (modelo, ano, placa, quilometragem).

Registro de revisões realizadas e futuras (tipo, data prevista, descrição).

Consulta de histórico de revisões por veículo.

Interface responsiva e intuitiva para facilitar a navegação.

Validação de dados no frontend e backend.

3. Restrições



Não será implementada autenticação avançada (login com OAuth, etc.).

Não haverá integração com APIs externas (ex.: fabricantes de veículos).

O sistema será desenvolvido para uso em navegadores desktop e mobile, mas sem otimização para aplicativos nativos.
